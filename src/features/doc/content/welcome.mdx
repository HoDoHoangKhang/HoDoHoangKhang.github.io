## Stack

- Next.js 16
- Tailwind CSS v4
- shadcn/ui

## Features

- Clean & modern design
- Light/Dark themes
- SEO optimized ([JSON-LD schema](https://json-ld.org), sitemap, robots)
- Installable as PWA
- Blog and docs powered by MDX

## Registry

Reusable components and blocks are available via the [shadcn CLI](https://ui.shadcn.com/docs/cli) registry.
