# Portfolio — Hồ Đỗ Hoàng Khang (Hoàng Khang / HK)

Personal portfolio of **Hồ Đỗ Hoàng Khang** (Hoàng Khang).

## Stack

- Next.js 16
- Tailwind CSS v4
- shadcn/ui

## Features

- Clean, responsive layout
- Light/Dark themes
- SEO (JSON-LD, sitemap, robots)
- Blog with MDX
- Section-based navigation with smooth scroll

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```
